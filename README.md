here I'll walk you through how I bioinformatically process 16S rRNA amplicon sequences using the [dada2 pipeline](https://benjjneb.github.io/dada2/index.html) in R

I used this script to analyze the microbial community composition of permafrost from an active thaw slump in the Inuvialuit Settlement Region of the Northwest Territories, Canada (preprint coming soon!)
